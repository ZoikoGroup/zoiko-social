05 — Planning Notes

  • Verified Advertising has no user-navigated destination — ads appear inline in the Feed, News, and Events pages with mandatory disclosure labels. Advertisers manage campaigns via a separate Admin portal, not the main nav.
  • Pet Diary and Health Passport are per-pet features, not top-level destinations. Access path: Left sidebar "My Pets" section on desktop, or Profile tab → My Pets on mobile. If a user has multiple pets, each gets its own diary and passport page.
  • Lost & Found lives in the tray on mobile but should push a badge notification on the tab bar icon when a nearby report matches the user's registered pets or species preferences.
  • Breeding Match is access-gated — the nav link is always visible, but listing creation is locked behind verified account + uploaded health certificates. Browsing approved listings requires only a login.
  • Vet Finder widget in the right sidebar is a 2-result preview. Tapping "See all" opens the full Vet Finder page in the center column, replacing the feed — the sidebar stays in place.
  • Left sidebar Quick Links use a collapsible "Show more / Show less" toggle on screens under 900px tall — prevents the sidebar from overflowing on laptops. First four links (News, Events, Adoption, Lost & Found) are always visible.
